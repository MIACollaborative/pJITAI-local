# mDOT Reinforcement Learning Cloud Service and Associated Interface Libraries

## API Routes
- Default get model: 
  * HTTP Method: `GET`
  * URL: `HOST/api/UUID/`
- Decision route: 
    * HTTP Method: `POST`
    * URL: `HOST/api/UUID/decision`
- Batch update: 
    * HTTP Method: `POST`
    * URL: `HOST/api/UUID/batch_update`
- Validate: 
    * HTTP Method: `POST`
    * URL: `HOST/api/UUID/validate`
