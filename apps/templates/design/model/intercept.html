The intercept refers to the intercept term in the linear regression model of the proximal outcome. The intercept corresponds to the value of the proximal outcome when all the covariates are zero.

Many papers may not provide the prior mean and the prior standard deviation of the intercept.

If you don’t have good information about the prior mean and prior standard deviation, then we suggest you set the prior mean to be 0 and the prior standard deviation to be 10. Please read [tutorial] for more details.

The prior of the intercept cannot be found from the HeartSteps V1 paper. Thus, we demonstrate using the default values here, i.e., prior mean = 0 and prior standard deviation = 10.
