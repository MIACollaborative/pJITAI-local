We use the attributes of the covariates to check if the API receives inappropriate/missing input from your mobile app. Please notice that if your covariate is binary, we ask you to describe how the values of 0 and 1 translate to the states an individual is in.

Please notice that if your covariate needs to be transformed, the transformation needs to be implemented in the mobile application; the algorithm will not do the transformation.

Please read [tutorial] for more details.

In HeartSteps, “Location” is coded as 1 when an individual is in an unknown location (neither at home nor at work) and zero otherwise.
