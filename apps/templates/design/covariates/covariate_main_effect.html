We are asking about the main effect of the covariate in the Bayesian linear regression. This coefficient corresponds to the amount of change in the proximal outcome per unit change of the covariate when intervention option B is provided (i.e., “do nothing” in the HeartSteps example) and all the other covariates are fixed.

Many papers may not provide the prior mean and the prior standard deviation of the main effect of the covariate.

If you don’t have good information about the prior mean and prior standard deviation, then we suggest you set the prior mean to be 0 and the prior standard deviation to be 10. Please read [tutorial] for more details.

This information is not reported in the HeartSteps V1 paper. Thus, we demonstrate using the default values here, i.e., prior mean = 0 and prior standard deviation = 10.
